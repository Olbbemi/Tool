	- GameTool: 파일이름 작성 후 (반드시 입력해야함) Add 버튼으로 저장하고 싶은 목록을 작성한 뒤 [최대 11개]
			Create 버튼누르면 표 생성, 표에 입력할 데이터 작성한 뒤 Save 버튼 누르면 저장

	 example: | Name    | position | HP | MP |
		  | monster |    5     | 10 | 20 |

	- SystemTool: 파일이름 작성 후 (반드시 입력해야함) Add 버튼누름(한번만), scope 입력후 마우스 우클릭하면 데이터 입력할 텍스트박스 생성
			좌측 텍스트박스는 변수명, 우측 텍스트박스는 값입력

			+ 버튼 누르면 새로운 스코프 생성, - 버튼 누르면 해당 스코프의 끝을 의미, Save 버튼 누르면 저장
			
	 example: :Scope
		  >
		  data1 = 10;
		  data2 = 20;
		  	:newScope
		 	>
		 	data3 = 30;
			data4 = 40;
		 data5 = 50;
